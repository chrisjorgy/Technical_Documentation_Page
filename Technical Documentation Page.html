About

HTML stands for hyper-text markup language. HTML is the framework of websites, and can be combined with CSS to add creative ideas to your websites. Javascript can also be used with HTML and CSS to add functionality and interaction on your web page. HTML has gone through numerous updates and versions. Currently HTML5 is the highest and most up to date version. In this lesson we will learn how to create a fully functioning website ready for CSS and Javascript. This is the beginning of your web developement adventure.

HTML Elements

HTML elements contain start tags and end tags, and are the building blocks of website creation. The h1 element is used for bold titles, with smaller and smaller headings as you get to h6. P is defined as the paragrpah element and is the portion of the website you are reading right now. Br stands for line break and can be thought of as if you were to hit the return key in a word document. You can use "br" to bring your code down a line or create a space. HTML defines the whole document and is needed to allow this code to function as HTML does. Body is the main body of work of an HTML page and contains all of the elements listed above, minus the HTML tag. Below you can see an example of Elements and a list of common elements.

Paragraphs

The parapgraph element is used to write large sections within a website. This element can be nested within many other elements and is very useful when trying to write large amounts of text. In fact you are currently reading the text nested with a opening and closing paragraph element. P elements are easily styled with CSS to change characteristics such as the font-family, text-align, text-decoration and many more. An example of CSS styling is p {text-decoration: underline;}.

Text Area

Text area can be nested within forms to allow the user to leave comments and/or create a space to write down information. The most common use of this would be a comment box or a text area on a forum or blog post such as twitter.

Placeholder Text

Placeholder text can be useful when used within forms. When a user is required to input an email, name, or any other information. A placeholder text can give a user an example of the text that is needed. Placeholder text immediatey disappears once the user begins typing. Placeholder text isnt necessary to have a fully functioning website, but can allow for a better user experience and can give the appearance of a more completed website. A placeholder can be styled and changed by using CSS values. It is possible to change the text style, color and font size by using the following syntax.
input::placeholder {color: red; font-size: 1.2em; font-style: italic;}

Checkboxes

Checkboxes are used to choose an option and are typically found on forms for surveys or personal information. A checkbox can allow for multiple checks, unlike the similar radio button. A checkbox can also be styled using CSS to change the font, checkbox type and overall appearance. An example of the CSS is label {font: 1rem 'Fira Sans', sans-serif;} and input {margin: .4rem;}.
